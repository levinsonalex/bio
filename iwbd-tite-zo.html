Overview

Three small flash games in one. A difficult tower-defense like mini game, which leads into the platformer game ending in a boss fight. And seperate from the main progression is a first person shooter survival mode.

I created this game during my free time outside of my first programming class in high school. It is very rough around the edges, but it really shows my enthusiasm for programming. I barely knew anything about coding, and the majority of this is self taught. There aren't even data structures involved, which led to this gigantic nested if statement block to set the z-coordinate of the four zombies in Zombie Onslaught.
